# 大屏幕相关接口
## 1.获取各部门办理时间及满意度一览
### 1.1 接口描述
| TYPE | URL                           | Description |
| ---- | ----------------------------- | ----------- |
| GET | /Display-Module/Summary/GetDataStat | 获取各部门办理时间及满意度一览  |

### 1.2 请求范例
无

### 1.3 返回范例
```javascript
[
      {
            "deptId": "1",
            "deptName": "不动产登记",
            "detailData": {
                  "dCallNum": 3,
                  "dHandleTime": 29,
                  "dHandleNum": 2,
                  "dHandledNum": 2,
                  "dHandlingNum": 1,
                  "dSatRate": 100,
                  "mCallNum": 28,
                  "mHandleTime": 20,
                  "mHandleNum": 25,
                  "mHandledNum": 25,
                  "mHandlingNum": 4,
                  "mSatRate": 100
            }
      },
]
```  

### 1.4 请求参数说明
无

### 1.5 返回参数说明
|参数名|类型|说明|
|:-----  |:-----|----- |	
|deptId|string|窗口ID，1=不动产，2=社保，3=医保，4=公积金，5=市民卡，6=交通|	
|deptName|string|窗口名称|
|detailData|data|窗口具体数据|
|dCallNum|int|日叫号量|
|dHandleTime|int|日办件时间|
|dHandleNum|int|日办件量|
|dHandledNum|int|日办结量|
|dHandlingNum|int|日在线受理量|
|dSatRate|int|日满意率|
|mCallNum|int|月叫号量|
|mHandleTime|int|月办件时间|
|mHandleNum|int|月办件量|
|mHandledNum|int|月办结量|
|mHandlingNum|int|月在线受理量|
|mSatRate|int|月满意率|

## 2.获取今日办件情况
### 2.1 接口描述
| TYPE | URL                           | Description |
| ---- | ----------------------------- | ----------- |
| GET | /Display-Module/Summary/GetHandleStat | 获取今日办件情况  |

### 2.2 请求范例
无

### 2.3 返回范例
```javascript
{
      "dCallNum": 3,
      "dHandledNum": 2,
      "dWaitNum": 20,
      "dWaitTime": 18,
      "dSatRate": 99,
      "deptStat": [
            {
                  "deptId": "1",
                  "deptName": "不动产登记",
                  "dNum": 2
            },
      ]
      "handleDetail": 
      [
            {
                  "handleId": "4",
                  "handleName": "综合受理（个人）",
                  "startTime": 1518230913524
            },
      ]
 }
```  

### 2.4 请求参数说明
无

### 2.5 返回参数说明
|参数名|类型|说明|
|:-----  |:-----|----- |	
|dCallNum|int|日叫号量|	
|dHandledNum|int|日办结量|
|dWaitNum|int|当前等待人数|
|dWaitTime|int|平均等待时间|
|dSatRate|int|日满意率|
|deptStat|list|部门分类|
|deptId|string|窗口ID|
|deptName|int|窗口名字|
|dNum|int|窗口日办件量|
|handleDetail|list|办理详情|
|handleId|string|办理ID|
|handleName|string|办理名称|
|startTime|long|办理开始时间|

无

### 5.3 返回范例
```javascript
[
      {
            "showId": "1",
            "name": "办理时间",

                  "dPeccancy": 29,
                  "dRealEstateReg": 17,
```  

## 5.获取风采第一屏
### 5.1 获取风采第一屏
| TYPE | URL                           | Description |
| ---- | ----------------------------- | ----------- |
| GET | /Display-Module/Summary/GetDataDisplayV2 | 获取风采第一屏  |

### 5.2 请求范例
无

### 5.3 返回范例
```javascript
[
      "handleStat": {
            "waitNum": 12,
            "waitTime": 13,
            "handleTime": 18
      },
      "callHandleStat": [
            {
                  "whichDay": -1,
                  "callNum": 968,
                  "HandleNum": 872
            },
            {
                  "whichDay": 0,
                  "callNum": 74,
                  "HandleNum": 62
            }
        ],
      "weeklyHandledNum": [
            {
                  "whichWeek": 1,
                  "wHandledNum": 0
            },
            {
                  "whichWeek": 2,
                  "wHandledNum": 1133
            }
        ],
        "dailyHandledNum": [
            {
                  "weekday": 1,
                  "dHandledNum": 872
            },
            {
                  "weekday": 2,
                  "dHandledNum": 62
            }
        ]
        "dSatRate": 99
    },
]
```  

### 5.4 请求参数说明
无

### 5.5 返回参数说明
|参数名|类型|说明|
|:-----  |:-----|----- |	
|handleStat|data|办件状态|	
|waitNum|int|当前等待人数|
|waitTime|int|平均等待时间|
|handleTime|int|平均办件时间|
|callHandleStat|list|叫号办件情况|
|whichDay|int|今日昨日，-1=昨日，0=今日|
|callNum|int|叫号量|
|HandleNum|int|办件量|
|weeklyHandledNum|list|本月周办件量统计|
|whichWeek|int|本月第几周，1=第一周，2=第二周，3=第三周……|
|wHandledNum|int|周办件量|
|dailyHandledNum|list|日办件量统计|
|weekday|int|本周周几，1=周一，2=周二|
|dHandledNum|int|日办件量|
|dSatRate|int|今日平均满意率|
